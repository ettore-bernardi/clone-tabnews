## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/clone-tabnews.git
   ```
2. Navigate to the project directory:
   ```bash
   cd clone-tabnews
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the development server:
```bash
npm start
```
